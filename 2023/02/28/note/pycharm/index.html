Pycharm Remote

Pycharm + Remote

Pycharm Professional

Download

需要下载 2021.02 版本或更低版本的 professional pycharm. 官网下载链接

Evaluate

  1. 下载安装后，会有30天的免费试用期。试用免费试用，进入pycharm professional。

  2. 点击 setting (快捷键: ctrl + atl + s)，进入插件(Plugins)，点击设置按钮，管理插件仓库(Manage Plugin Repositories)。

在仓库源中，加入 https://plugins.zhile.io

  1. 加入源完成后，在市场(Marketplace)中，搜索 IDE Eval Reset，并下载。

  1. 设置免费试用重置，reload和reset，即可重置免费期限。
    ！！！注意：超过30天未登陆，可能无法启动重置！！！

Remote 远程

Path of Server Interpreter

需要确认远程服务器的python环境，若服务器是anaconda环境，则需要寻找到 ./anaconda/bin/python3.X.exe 的路径(path)

New Project

点击新建项目(New Project), 选择纯python项目(Pure Python)。新建python项目

Interpreter

解释器是指 python 的解释器环境。若未配制远程解释器地址，则点就…，进行新建配制。

  1. 选择 SSH Interpreter， 在New server configuration中，填写服务器地址(Host)、端口(Port)和用户名(Username)

  2. 下一步需要输入用户名密码

  3. 再选择python环境地址，在前序中提及的path。

  4. 即可完成解释器配制。

Remote project location

配制远程路径(Remote project location)，此路径为python项目在远程服务器中的地址路径。

Location

配制本地项目路径(Location)，此路径为python项目在本地电脑中的地址路径。

Deployment

在顶部菜单栏中，选择工具(Tools) -> 部署(Deployment)，可以选择自动上传代码(Automatic Upload)——即实时传输本地python代码至远程服务器更新。

也可在Deployment中，选择Browse Remote Host，即可在右侧面板查看远程路径情况。项目所在目录，会有绿色背景高亮提示。若存在问题，可通过上侧 … ，对远程配制信息进行查看和修改。

也可以针对单个文件/文件夹，右击选择上传(Upload to @..)，达到随时上传更新文件。
